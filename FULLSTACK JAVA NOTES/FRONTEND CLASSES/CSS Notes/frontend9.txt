CSS overflow property
========================
The overflow property specifies what should happen if content overflow.

This property spcifies weither to clip content or to add scrollbars  when an element content is too big to fit in a specified area.

Note:
---------
The overflow property works for block elements with  a specified height.

value		Description
---------	-----------------
visible		The overflow is not clipped.It rendered outside the element 's box and 
		it is default value.

hidden		the overflow is clipped  and rest of the content will be invisible.

scroll		The overflow is clipped,but a scroll-bar is added to see the rest of the 
		content .

auto		The overflow is clipped,a scroll-bar should be added to the rest of the 
		content.

------------------

<html>
	<head>
		<title>MyPage!</title>
		<style>

			{
				width:200px;

				border:2px solid black;
				overflow:visible;
			}
		</style>

CSS border-radius property 
==========================
The border-radius property defines  the radius of the element's corners.

This property allows us to add rounded borders to elements.

This property can have from one to four values.
ex:
	border-top-left-radius
	border-top-right-radius
	border-bottom-right-radius
	border-bottom-left-radius
	border-radius shorthand 

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			div
			{
				width:200px;
				height:200px;
				margin:100px auto;
				border:2px solid black;
				overflow:auto;
				border-top-left-radius : 5px;
				border-top-right-radius: 10px;
				border-bottom-right-radius: 15px;
				border-bottom-left-radius: 20px;
			}
		</style>

CSS image shadow property
===========================
A drop shadow is effectively a blurred, offset version of the input image's alpha mask, drawn in a specific color and composited below the image.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			img
			{
				width:300px;
				height:300px;
				margin: 80px 420px;
				filter : drop-shadow(20px 50px #C3C3C3);
			}
		</style>
	</head>
	<body>
			<img src="images/micky.png" alt="No Image Found"/>
	</body>
</html>

Types of selectors in CSS
============================
We have five selectors in CSS.

1) Element Selector 

2) Group Selector 

3) Id Selector 

4) Class Selector 

5) Universal Selector 

1) Element Selector 
--------------------
The element selector selects HTML elements based on element name.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				color:#0000FF;
				background-color:#00ff00;
				text-align:center;
			}
		</style>
	</head>
	<body>
			<h1> Heading Tag </h1>
			<h1> Heading Tag </h1>
	</body>
</html>



2) Group Selector 
-----------------
The grouping selector selects all the HTML elements with the same style definitions.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1,p,div
			{
				color:#0000FF;
				background-color:#00ff00;
				text-align:center;
			}
		</style>
	</head>
	<body>
			<h1> Heading Tag </h1>
			<p> Paragraph Tag </p>
			<div> Division Tag </div>

3) Id Selector 
---------------
The id selector uses the id attribute of an HTML element to select a specific element.

The id of an element is unique within a page, so the id selector is used to select one unique element.

To select an element with a specific id, write a hash (#) character, followed by the id of the element.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			#myId
			{
				color:#FF0000;
				text-align:center;
				background-color:#FFFF00;
			}

		</style>
	</head>
	<body>
			<h1 id="myId"> Heading Tag </h1>
			<p> Paragraph Tag </p>
			<div> Division Tag </div>
	</body>
</html>

4) Class Selector
------------------
The class selector selects HTML elements with a specific class attribute.

To select elements with a specific class, write a period (.) character,
followed by the class name.

ex:
---

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			
			.myClass
			{
				color:#FF0000;
				text-align:center;
				background-color:#FFFF00;
			}
		</style>
	</head>
	<body>
			<h1 class="myClass"> Heading Tag </h1>
			<p > Paragraph Tag </p>
			<div > Division Tag </div>
	</body>
</html>

5) Universal Selector 
----------------------
The universal selector (*) selects all HTML elements on the page.


ex:
---

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			
			*
			{
					color:blue;
					background-color:cyan;
					text-align:center;
			}
		</style>
	</head>
	<body>
			<h1> Heading Tag </h1>
			<p > Paragraph Tag </p>
			<div > Division Tag </div>
	</body>
</html>
