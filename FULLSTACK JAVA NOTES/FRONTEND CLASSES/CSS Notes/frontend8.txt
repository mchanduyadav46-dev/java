CSS Border Property 
===================
It is used to specify style, color and width of HTML element border.

We have following list of CSS border properties.

ex:
	border-style 
	border-color
	border-width 
	border shorthand 
ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				border-style:solid;
			}
			h2
			{
				border-style:dotted;
			}
			h3
			{
				border-style:dashed;
			}
			h4
			{
				border-style:double;
			}
			
		</style>
	</head>
	<body>
		<h1> Heading tag 1 </h1>
		<h2> Heading tag 2 </h2>
		<h3> Heading tag 3 </h3>
		<h4> Heading tag 4 </h4>
	</body>	
</html>

CSS margin property 
===================
CSS margin property is to specify space around the elements (outside the defined border).

We have following list of CSS margin properties.

ex:
	margin-top 
	margin-right 
	margin-bottom 
	margin-left 
	margin shorthand 

All the above margin property contains following values.

ex:
	Value			Description 
	-----			-----------
	auto 			Browser calculates the margin 

	length 			It specifies a margin in px, pt, cm, em, etc.	

	Percentage 		It specifies a margin in percent(%).

	Inherit 		It specifies a margin from parent element.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				border:2px solid black;
				margin-top:100px;
				margin-right:100px;
				margin-bottom:100px;
				margin-left:100px;
			}
		</style>
	</head>
	<body>
			<h1> Heading Tag </h1>
	</body>
</html>

CSS height and width property 
=============================
CSS height property is used to set the height of an element.

CSS width property is used to set the width of an element.

CSS height and width do not include border, margin and padding.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			div
			{
				border:2px solid black;
				width:200px;
				height:200px;
			}
		</style>
	</head>
	<body>
			<div></div>
	</body>
</html>

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			div
			{
				border:2px solid black;
				width:200px;
				height:200px;
				margin:150px auto;
			}
		</style>
	</head>
	<body>
			<div></div>
	</body>
</html>
