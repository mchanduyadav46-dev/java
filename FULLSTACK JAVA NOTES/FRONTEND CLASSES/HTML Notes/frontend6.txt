Graphics in HTML
================
There are two ways to draw the graphics in HTML.

1) SVG
2) CANVAS 

1) SVG
-------
SVG stands for Scalable Vector Graphics.

It is used to draw two dimensional based vector graphics in HTML.

World Wide Web Consortium (W3C) prefers SVG technology to draw the graphics in HTML.

A <svg> is a container tag.

It contains various methods to display circle, rectangle, polygon, graphic image and etc.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<svg style="border:2px solid black;width:400px;height:400px;">
				
			</svg>
	</body>	
</html>

ex:
----

				<circle cx="200" cy="200" r="80" 
						fill="#FFFF00" stroke="#FF0000" stroke-width="5px"/>
			</svg>

2) CANVAS 
----------
It is used to draw the graphics via javascript.

A <canvas> is a container tag.

It contains various methods to draw the shapes like lines, circle, rectangle, graphic images, polygon and etc. 
		 
ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<canvas id="myId" style="border:2px solid black;width:400px;height:400px;">
				
			</canvas>
			
			<script>
				var context = document.getElementById("myId");
				var c = context.getContext("2d");
				c.beginPath();
				c.moveTo(0,0);
				c.lineTo(800,400);
				c.stroke();
			</script>
	</body>	
</html>

ex:
----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<canvas id="myId" style="border:2px solid black;width:400px;height:400px;">
				
			</canvas>
			
			<script>
				var context = document.getElementById("myId");
				var c = context.getContext("2d");
				c.beginPath();
				c.arc(100,100,50,0,2*Math.PI);
				c.stroke();
			</script>
	</body>	
</html>

HTML audio tag 
===============
The HTML <audio> tag/element is used to play an audio file on a web page.

<audio> tag contains "controls" attribute adds audio control like play,pause ,volume and etc.

The <source > tag/element allows us to specify alternate audio file which the
browser my choose.

HTML audio formats can be MP3,WAV,OGG and etc.

HTML Audio Media types are 

File Formats		Media Type
------------		----------
MP3			audio/mpeg

OGG			audio/ogg

WAV			audio/wav

and etc
 

ex:
----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<audio src="media/song.mp3" controls>
			</audio>
	</body>	
</html>

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<audio  controls>
				<source src="media/song.mp3" type="audio/mpeg"/>	
			</audio>
	</body>	
</html>

ex:
---

<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<audio controls muted >
				<source src="media/song.mp3" type="audio/mpeg"/>	
			</audio>

HTML video tag 
===============
The <video> tag or element is used to embed video content in a document such as
movie clip,other video streams.

The <video> tag contains one or more <source> tag with different video source.

There are three supported video fromats in html are MP4,webM and OGG.

HTML video media types are 

File format		Media Type
-----------		-----------
MP4			video/mp4

OGG			video/ogg

and etc.
