Q) What is the difference between <b> tag and <strong> tag?

<b> tag 
-------
It is used to display the text in bold without importance.

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<b>Bold Tag</b>
	</body>
</html>

<strong>
--------
It is used to display the text in bold with importance.

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<strong>Strong Tag</strong>
	</body>
</html>

Phrase Tags 
==============
Phrase tags are special purpose tags they defined special meaning to the text or block of text.

We have following list of phrase tags in HTML.

1) Abbreviation - <abbr>
2) Keyboard     - <kbd>
3) address      - <address>
4) short quote  - <q> 
5) Strike       - <s> 
6) variable     - <var>
7) sample       - <samp>
8) blockquote   - <blockquote> 
9) cite		- <cite>
10) mark	- <mark> 
11) subscript   - <sub>
12) superscript   - <sup> 
13) bi directional order - <bdo> 
14) code        - <code>
15) Preformatted tag - <pre> 
and etc.

HTML image Tag 
===============
A <img> tag is used to display the images in a web page.

It it is a opening tag and does not have any closing tag.

It contains only attributes as given below.

ex:
	Attribute		Description 
	----------		-----------
	src			It is used to locate an image
	width 			It is used to adjust the width of an image
	height 			It is used to adjust the height of an image 
	alt 			It is used to display alternate msg if image is not found

We have following list of images.
ex:
	File Format	Abbreviation 
	----------	-------------
	JPEG		Joint Photographic Expert Group
	PNG 		Portable Network Graphics	
	SVG		Scalable Vector Graphics
	GIF		Graphical Interchange Format.
	and etc.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
		<img src="images/rock.jpg" width="200px" height="200px"/>	
	</body>
</html>

ex:
---
